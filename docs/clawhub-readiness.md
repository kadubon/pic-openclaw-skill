# ClawHub / Registry Readiness

This repository is prepared for manual GitHub-based installation. It is not automatically published to ClawHub.

ClawHub or registry submission should happen only after verifying current registry requirements. If a registry requires a different license for skill text, make an explicit license decision first.

The skill must remain no-auto-clone, no-auto-install, and no-action-execution.

## Checklist

- [ ] verify current registry submission requirements
- [ ] verify license requirements
- [ ] verify root `SKILL.md` is accepted
- [ ] verify nested skill directory path if supported
- [ ] confirm no install scripts are required
- [ ] confirm no secret, credential, or local path appears in examples
- [ ] confirm manual install works
