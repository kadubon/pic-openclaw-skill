# Skill-Only Mode

Skill-only mode uses only the OpenClaw skill instructions and deterministic checklist.

Before external-effect actions, OpenClaw should create an action proposal, list evidence, list missing evidence, list rollback, classify risk, and preserve unresolved items.

Skill-only mode does not require PIC, Python, uv, network access, or a PIC clone. It also does not replace OpenClaw sandboxing, approval prompts, tool policy, workspace-only filesystem controls, or skill allowlists.

Generated agent output is a candidate, not verified work.
